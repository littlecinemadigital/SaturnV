# C/C++ Styleguide.

## Table of Contents

  ## Headers

 Generally speaking, all `.c` and `.cpp` files should be accompanied by a `.h` or `.hpp` file. 
 
 ### Self Contained Headers
 
 Header files should be self contained and suffixed in `.h` if they're C and `.hpp` if they're C++.
